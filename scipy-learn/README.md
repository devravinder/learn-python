## Project Setup

1. `uv init scipy-learn && cd scipy-learn`
2. `uv add scipy numpy matplotlib`
3. for jupyterlab
   `uv add jupyterlab`
   `source .venv/bin/activate`
   `jupyter lab`
   `deactivate`
