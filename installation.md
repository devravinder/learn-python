# Installation

1. Install for linux

    ```bash
    sudo apt update
    sudo apt install python3 # python
    sudo apt install python3-pip  # package manager

    ```

    add `alias py="python3"` for quick use

2. Check python installation

    ```bash
    python3 --version
    ```

3. Running a python script

   ```bash
     python3 hello.py
     
   ```
