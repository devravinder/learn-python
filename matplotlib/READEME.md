# Pandas

## Set up

### From VSC
  >
  > Note: open only this folder in vsc

1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip install pandas numpy matplotlib`
    - if needed install `pyplot` seperately
4. `pip install ipykernel` to run .ipynb
5. `pip freeze > requirements.txt`
6. run `python3 main.py` or use `.ipynb` files
7. `deactivate`
