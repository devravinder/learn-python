# NumYp = Numerical Python

## Setup

### with pycharm

1. open with pycharm
2. `pip freeze > requirements.txt`
3. usage `pip install -r requirements.txt`

### With VSC

1. `python3 -m venv venv`
2. `source venv/bin/activate`
3. `pip freeze > requirements.txt`
4. run `python3 main.py`
5. `deactivate`
