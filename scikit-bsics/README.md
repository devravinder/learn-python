## Set up

1. `uv init scikit-basics && cd scikit-basics`
2. `uv add numpy pandas matplotlib scikit-learn jupyterlab`
3. `source .venv/bin/activate` if we are using jupyterlab
4. `jupyter lab`
5. `deactivate`
